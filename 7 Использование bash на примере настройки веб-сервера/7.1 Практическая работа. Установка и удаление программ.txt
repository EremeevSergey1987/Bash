7.1 Практическая работа. Установка и удаление программ

apt-get специальная программа с помощью которой можно устанавливать и удалять пакеты
apt-get update такая команда как раз обновит индекс пакетов. Индекс пакетов это все пакеты для данной операционной системы
apt-get install ms - так устанавливаем программу с именем ms
apt-get upgrate ms - так обновляем конкретный пакет. В данном случае пакет ms
apt-get upgrate - обновать все установленные пакеты. ОПАСНО! РАСНОЕ ПО МОЖЕТ ПЕРЕСТАВТЬ РАБОТАТАТЬ ДРУГ С ДРУГОМ
apt-get remove ms - так мы удаляем конкретную программу
