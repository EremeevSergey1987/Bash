6.3 Условия и логические выражения

За проверку условий отвечает оператор if

//////////////////
if [ "$1" != "" ]
then
    echo "YES"
else
    echo "NO"
fi
//////////////////

//////////////////
-eq равно
-gt более чем. Вот такой вот оператор сравнения...
-ne не равно)))
-ge больше либо равно
-lt меньше
-le меньше либо равно

Файловые операторы
if [ -f filename] файл существует
if [ -d filename] директория существует
if [ -r filename] проверка существования и доступности для записи
if [ -w filename] проверка существования и доступности для чтения
if [ file1 -nt file2 ] проверка даты изменения файла


if [ $# -gt 0 ]
then
    echo "YES"
else
    echo "NO"
fi
