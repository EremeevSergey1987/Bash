5.1 Практическая работа. Параметры и настройки bash

which ls - так можно посмотреть где лежит команда ls
which php

set - посмотреть все установленные переменные окружения
set | less все установленные переменные окружения
echo $EDITOR - посмотреть какой у нас редактор по умолчанию
export EDITOR=vi - назначить редактор по умолчанию
export EDITOR='' удалить значение переменной
