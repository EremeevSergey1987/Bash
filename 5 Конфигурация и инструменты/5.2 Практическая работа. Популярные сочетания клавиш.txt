5.2 Практическая работа. Популярные сочетания клавиш

ctrl + a переход к началу строки
ctrl + e переход к концу строки
ctrl + c прерывание процесса
ctrl + z приостановка процесса
ctrl + d завершение сессии
ctrl + l команда очистки экрана терминала. При этом всё напечатанное уходит наверх
ctrl + k очищает все что справа до конца строки
ctrl + u очищает все что слева до начала строки
ctrl + r поиск по истрии запросов
ctrl + j прекращает поиск
ctrl + -> двигаем курсор по словам в право
ctrl + <- двигаем курсор по словам в лево
